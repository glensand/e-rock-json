template<typename TValue >
auto erock::load ( std::string_view  json)

Tries to parse JSON string, if succeeded then stores all the loaded values to the related fields of the structure. The type of the structure is required, obviously;.

Parameters
jsonString with json to be parsed;
TValue- Type of the structure which fields matches the objects from JSON file. Each field ot the TValue have to be an instance of the erock::field structure with proper name. All these fields are required in the file, if the attribute of the object is not present in the file, the default value will be stored to the desired position.
Exceptions
std::runtime_errorwith description of the occurred error (Invalid value, not all braces are placed right e.t.c)
